# tmux-huma Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** A single zero-dependency Rust binary + TPM plugin that provides tmux status-bar awareness widgets — online, mode badge, battery, load — replacing tmux-online-status and tmux-prefix-highlight.

**Architecture:** Mirror tmux-anka. A background `huma daemon` writes `@huma_online`/`@huma_battery`/`@huma_load` tmux user options (non-blocking); the `mode` badge is a static tmux format string from `huma mode`. Per-widget subcommands (`huma online|battery|load`) print one value (debug/test) and the daemon reuses the same functions.

**Tech Stack:** Rust (clap, anyhow), tmux CLI, `/proc`, `/sys`, std `TcpStream`.

## Global Constraints

- Zero runtime dependencies; deps limited to `clap`, `anyhow`. No async runtime.
- Linux-only (`/proc`, `/sys`).
- Every config option read via `tmux show-options -gqv`, with a default; names are `@huma-*`. Output options are `@huma_online`, `@huma_battery`, `@huma_load`, `@huma_mode`.
- Theme-agnostic: emit value + icon only; never emit colour escapes.
- Build/test offline: `cargo test --offline`; lint `cargo clippy --offline --all-targets` must be warning-clean.
- Version bumps land in both `Cargo.toml` and `Cargo.lock`.
- Canonical repo: `~/.kod/tmux/tmux-huma` (push over SSH `git@github.com:kenanpelit/tmux-huma.git`); `.cachy` embeds it as a gitlink at `modules/tmux/dotfiles/tmux/plugins/tmux-huma`.

```rust
//! Thin wrapper around the `tmux` CLI.
use anyhow::{bail, Context, Result};
use std::process::Command;

pub fn run(args: &[&str]) -> Result<String> {
    let out = Command::new("tmux").args(args).output().context("spawn tmux")?;
    if !out.status.success() {
        bail!("tmux {} failed: {}", args.join(" "), String::from_utf8_lossy(&out.stderr).trim());
    }
    Ok(String::from_utf8_lossy(&out.stdout).trim_end_matches('\n').to_string())
}
pub fn run_ok(args: &[&str]) {
    let _ = Command::new("tmux").args(args).output();
}
pub fn server_running() -> bool {
    Command::new("tmux").arg("list-sessions").output().map(|o| o.status.success()).unwrap_or(false)
}
pub fn global_option(name: &str) -> String {
    run(&["show-options", "-gqv", name]).unwrap_or_default()
}
pub fn set_global_option(name: &str, value: &str) {
    run_ok(&["set-option", "-g", name, value]);
}
```

```rust
pub fn mem_used_pct(meminfo: &str) -> Option<u32> {
    let mut total = 0u64;
    let mut avail = 0u64;
    for line in meminfo.lines() {
        if let Some(v) = line.strip_prefix("MemTotal:") {
```

```rust
//! Update daemon: periodically write the widget user options. Single-instance.
use anyhow::{bail, Result};
use std::fs;
use std::path::{Path, PathBuf};
use std::thread;
use std::time::Duration;

use crate::config::Config;
use crate::{battery, load, online, tmux};

pub fn once(cfg: &Config) -> Result<()> {
    if !tmux::server_running() {
        bail!("no tmux server running");
    }
    tmux::set_global_option("@huma_online", &online::widget(cfg));
    tmux::set_global_option("@huma_battery", &battery::widget(cfg));
    tmux::set_global_option("@huma_load", &load::widget(cfg));
    Ok(())
}

pub fn run(cfg: &Config) -> Result<()> {
    let lock = lock_path();
    if another_running(&lock) {
        return Ok(());
    }
    write_pid(&lock);
    let interval = interval(cfg);
    loop {
        let _ = once(cfg);
        thread::sleep(interval);
        if !tmux::server_running() {
            break;
        }
    }
    let _ = fs::remove_file(&lock);
    Ok(())
}

fn interval(cfg: &Config) -> Duration {
    if let Ok(ms) = std::env::var("HUMA_DAEMON_INTERVAL_MS") {
        if let Ok(ms) = ms.parse::<u64>() {
            return Duration::from_millis(ms);
        }
    }
    Duration::from_secs(cfg.interval_secs)
}

fn lock_path() -> PathBuf {
    let dir = std::env::var("XDG_RUNTIME_DIR").unwrap_or_else(|_| "/tmp".into());
    PathBuf::from(dir).join("huma.daemon.lock")
}

fn another_running(lock: &Path) -> bool {
    fs::read_to_string(lock).ok()
        .and_then(|s| s.trim().parse::<i32>().ok())
        .is_some_and(|pid| Path::new(&format!("/proc/{pid}")).exists())
}

fn write_pid(lock: &Path) {
    let _ = fs::write(lock, std::process::id().to_string());
}
```
Note: the lockfile lives in `XDG_RUNTIME_DIR`; the integration test overrides it (below) by pointing `XDG_RUNTIME_DIR` at the throwaway temp dir.

```rust
pub fn has_tmux() -> bool {

}
fn tmux_raw(socket: &str, args: &[&str]) -> Output {
    Command::new("tmux").args(["-f", "/dev/null", "-L", socket]).args(args).output().expect("spawn tmux")
}
pub fn tmux(socket: &str, args: &[&str]) -> String {
    let o = tmux_raw(socket, args);
    assert!(o.status.success(), "tmux -L {socket} {args:?}: {}", String::from_utf8_lossy(&o.stderr));
    String::from_utf8_lossy(&o.stdout).trim_end_matches('\n').to_string()
}
pub fn wait_for_exit(child: &mut Child, timeout: Duration) -> bool {
    let start = Instant::now();
    while start.elapsed() < timeout {
        match child.try_wait() { Ok(Some(_)) => return true, Ok(None) => std::thread::sleep(Duration::from_millis(50)), Err(_) => return true }
    }
    let _ = child.kill();
    false
}

        let uniq = std::time::SystemTime::now().duration_since(std::time::UNIX_EPOCH).unwrap().as_nanos();
```
